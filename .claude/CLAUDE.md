## Tool Know-How

Do not assume — verify. When looking up tool capabilities, use sources in this order:

1. **Local code and docs** — `README`, `/docs`, type definitions, existing code examples
2. **Local CLI help** — `tool --help`, `man tool`
3. **Official docs** (Context7 MCP)
4. **Official source repository** — GitHub or GitLab repo, source code, release notes
5. **Community or forums** — Stack Overflow, GitHub Issues — only as a last resort for undocumented errors or bugs
