## Tool Know-How

Do not assume — verify. When researching tool capabilities, use sources in this order:

1. **Local code and docs** — `README`, `/docs`, type definitions, existing code examples
2. **Local CLI help** — `tool --help`, `man tool`
3. **Official web documentation** — strictly official docs, preferring Context7 MCP
4. **Official source repository** — the tool's GitHub/GitLab repo, source code, and release notes
5. **Community/forums** — Stack Overflow, GitHub Issues, forums — only as a last resort for undocumented errors or bugs
